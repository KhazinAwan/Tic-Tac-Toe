# Tic Tac Toe

A browser-based Tic Tac Toe game built as part of **The Odin Project – Intermediate JavaScript** course. This project focuses on applying JavaScript modules, factory functions, DOM manipulation, and event-driven programming to build a complete interactive game using only vanilla JavaScript.

---

## Features

* Two-player local gameplay
* Custom player names
* Selectable player markers
* Active player turn indicator
* Dynamic game status updates (current game result, win, or tie)
* Win detection for rows, columns, and diagonals
* Tie detection
* Winning combination highlighting
* Restart game functionality
* Input validation for player names and marker selection
* Adaptive sizing using CSS Grid, Flexbox, and `clamp()`

## How to Play

1. Click **Start Game**.
2. Enter Player 1's name and choose a marker.
3. Repeat the setup for Player 2.
4. Players take turns placing their markers on the board.
5. The first player to align three matching markers horizontally, vertically, or diagonally wins.
6. If all cells are filled without a winner, the game ends in a tie.
7. Click **Restart Game** to begin a new game.
